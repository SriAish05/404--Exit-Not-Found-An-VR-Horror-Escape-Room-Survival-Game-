# 404: Exit Not Found - VR Horror Escape Room 
 
A VR horror escape room built using Unity and Meta Quest 3. 
 
## Features 
- AI ghost behavior (patrol, chase, attack) 
- Interactive puzzle system 
- Realistic lighting and 3D spatial audio 
 
## Tech Stack 
- Unity (HDRP) 
- C# 
- XR Interaction Toolkit 
- NavMesh AI
